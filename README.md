# Miami Rental Mafia — Demo Site

Premium exotic & luxury car rental landing page (demo). Single static folder — no build step.

- **`index.html`** — the entire site (HTML/CSS/JS, images inlined as base64)
- **`hero.mp4`** — cinematic hero background video
- **`logo.png`** — transparent brand logo

## Features
- Cinematic video hero, floating glass header
- Navy/blue theme, glowing animated cards, scroll pop-in text
- Pinned "stop-scroll" car showcase
- On-site reservation modal + concierge chatbot (demo only — no real bookings)

## Deploy
It's a static site. Any of these work with **zero config**:
- **GitHub Pages** — Settings → Pages → Deploy from branch `main` / root
- **Netlify** — "Add new site → Import from Git", publish directory = repo root
- **Vercel** — "Add New → Project → Import", framework preset = *Other*

Every push to `main` auto-deploys, so the live site updates as the project is edited.
